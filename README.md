# Chord Chart

This project converts a markup language into a chord chart.

```
# Untitled
## Artist

(Intro)
; 6/8 Ab . ; Asus4 . [ Ab E/A ; D/A . ]

(Verse 1)
[ A C#m(b5)/G# ; F#m7  C#m ; D .   ; Esus4 E ]

(Bridge 1)
; C#m7 .   ; F#m7  .   ; D E   ; A . ;

(Chorus 1)
[ A   .    ; Asus4 .   ; A E/A ; D/A . ]

(Outro)
; D  . ; A . ;;.
```
